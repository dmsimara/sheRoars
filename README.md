
# Civic Issue Reporting App

A mobile application built using **Flutter** that empowers citizens to report local civic issues directly to the concerned authorities.

## 📱 Features

- 📝 Report a wide range of civic issues (e.g., road damage, garbage collection, street lights)
- 📍 Pin issue locations using an interactive map
- 🧾 Track report statuses
- 🔐 Authentication for registered users
- 👤 Guest reporting without login
- 📊 Admin dashboard to manage and resolve reports

## 🛠 Tech Stack

- **Flutter** – Frontend UI
- **Firebase** – Backend services (Authentication, Firestore, Storage)
- **Google Maps API** – Location-based reporting

## 🧑‍💻 How to Run

1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/your-repo-name.git
   ```

2. Navigate to the project folder:
   ```bash
   cd your-repo-name
   ```

3. Install dependencies:
   ```bash
   flutter pub get
   ```

4. Run the app:
   ```bash
   flutter run
   ```

> 💡 Make sure you have Flutter and Dart installed in your environment.
